# 小灵通微信小程序

## 📋 项目简介

小灵通是一个工厂贸易供需对接平台，提供外贸采购、工厂库存、临时用工、代加工服务等功能。

## 🎯 项目概况

- **技术栈**: 微信小程序 + NestJS + MySQL + Redis
- **项目规模**: 72个API接口，30+张数据库表，40-50个页面
- **预计工期**: 50-74天（约2-3个月）
- **当前状态**: Phase 2 完成 - 架构设计与管控计划已完成

## 🔑 核心特性

### 业务特性
- ✅ 双用户体系（企业/临工完全隔离）
- ✅ 临工用工完整闭环（发布→报名→分配→打卡→结算）
- ✅ 多元化收费模式（会员、查看机会、广告、置顶、用工抽成）
- ✅ 信用分自动计算系统
- ✅ 查看机会FIFO消耗机制

### 技术特性
- ✅ 微信原生小程序
- ✅ NestJS模块化架构
- ✅ TypeORM + MySQL
- ✅ Redis缓存优化
- ✅ 微信支付集成
- ✅ 地理位置计算
