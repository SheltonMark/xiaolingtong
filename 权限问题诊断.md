1. 访问: https://github.com/settings/keys
2. 使用 `SheltonMark` 账号登录
3. 添加SSH公钥:
```
ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIDNv0GYaJLpOegU9sGuQ5Tzr/GqjarPHVT1GxU8aKbxO xiaolingtong@github.com
```
4. 告诉我 "已添加到SheltonMark账号"

### 方案2: 添加协作者权限

如果 `yichuan-zheng` 是你，`SheltonMark` 是别人:
1. 让 `SheltonMark` 访问: https://github.com/SheltonMark/xiaolingtong/settings/access
2. 点击 "Add people"
3. 添加 `yichuan-zheng` 为协作者
4. `yichuan-zheng` 接受邀请
5. 告诉我 "已添加为协作者"

### 方案3: Fork仓库

如果你想在自己的账号下工作:
1. 访问: https://github.com/SheltonMark/xiaolingtong
2. 点击右上角 "Fork"
3. Fork到 `yichuan-zheng` 账号
4. 修改远程URL:
```bash
git remote set-url origin git@github.com:yichuan-zheng/xiaolingtong.git
```
5. 推送到自己的仓库

### 方案4: 手动上传（最简单）⭐⭐⭐⭐⭐

不管账号问题，直接手动上传:
