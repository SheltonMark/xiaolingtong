## 🚀 快速开始

### 1. 准备环境
```bash
cd server
npm install
npm run test:setup
```

### 2. 启动MySQL
```bash
net start MySQL80
```

### 3. 运行测试
```bash
# 运行所有测试
npm run test:all

# 或分别运行
npm run test:integration              # 后端测试
npx playwright test                   # E2E测试
```

### 4. 查看报告
```bash
npm run test:coverage                 # 覆盖率报告
npx playwright show-report            # E2E报告
```
