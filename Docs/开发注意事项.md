## 开发阶段注意

### 微信开发者工具
- 开发阶段需勾选「不校验合法域名」才能用 IP 地址调试
- 真机调试需要开启「开发版」或用内网穿透

### 后端调试
- 后端开发模式下 wx.login 的 code 会走 mock 逻辑，自动创建测试用户
- 查看后端日志：`pm2 logs xiaolingtong-api`
- 重启后端：`pm2 restart xiaolingtong-api`

### 数据库
- 3306 端口未对外开放，只能 SSH 到服务器后本地连接
- 连接命令：`mariadb -u xlt -pXLT2026db xiaolingtong`

## 已知限制

- 聊天 WebSocket 在开发者工具中可能不稳定，建议真机测试
- 图片上传目前存本地 `/uploads` 目录，服务器重装会丢失
- 灵豆充值目前是模拟到账，未对接真实支付
