### 服务器部署流程

#### 连接方式
- 本地 `~/.ssh/config` 已配置 `xiaolingtong` 别名，直接执行：
  ```bash
  ssh xiaolingtong
  ```
- 服务器 IP / 用户名等信息见 `~/.ssh/config` 中 Host xiaolingtong 段

#### Git 拉取（SSH 协议）
- 服务器已配置 SSH 密钥对连接 GitHub（ed25519），无需输密码
- remote 地址：`git@github.com:SheltonMark/xiaolingtong.git`
- 如果 git pull 卡住（HTTPS 在国内服务器不稳定），确认 remote 是 SSH 协议：
  ```bash
  cd /root/xiaolingtong
  git remote -v
  # 如果还是 https，切换为 SSH：
  git remote set-url origin git@github.com:SheltonMark/xiaolingtong.git
  ```

#### 部署步骤（一键版）
```bash
ssh xiaolingtong "cd /root/xiaolingtong && git fetch origin && git reset --hard origin/main && cd server && npm run build && pm2 restart xiaolingtong-api"
```

#### 部署步骤（分步版）
```bash
# 1. 登录服务器
ssh xiaolingtong

# 2. 拉取最新代码
cd /root/xiaolingtong
git fetch origin
git reset --hard origin/main

# 3. 构建后端
cd server
npm install        # 有新依赖时执行
npm run build      # TypeScript 编译

# 4. 重启服务
pm2 restart xiaolingtong-api

# 5. 验证
pm2 list                      # 确认 status 为 online
pm2 logs xiaolingtong-api     # 查看启动日志，确认无报错
```

#### 常用运维命令
| 命令 | 说明 |
|------|------|
| `pm2 list` | 查看进程状态 |
| `pm2 logs xiaolingtong-api` | 查看实时日志 |
| `pm2 logs xiaolingtong-api --lines 100` | 查看最近100行日志 |
| `pm2 restart xiaolingtong-api` | 重启服务 |
| `pm2 stop xiaolingtong-api` | 停止服务 |
| `mariadb -u xlt -pXLT2026db xiaolingtong` | 连接数据库 |
