# 管理后台模块

## 1. 模块概述

管理后台是独立的Web端系统（PC端），供平台管理员使用。

**访问方式**：独立域名，浏览器访问

---

```
┌─────────────────────────────────────────────────────────────┐
│  人员分配 - 订单号：20260120                                 │
├─────────────────────────────────────────────────────────────┤
│  用工需求：组装工 - xxx工厂                                  │
│  需求人数：10人                                              │
│  报名人数：15人                                              │
│  抽成比例：20% [修改]                                        │
├─────────────────────────────────────────────────────────────┤
│  报名列表                                                    │
│  ☐ │ 姓名 │ 信用分 │ 接单数 │ 距离  │ 管理员资格 │ 操作     │
│  ──┼──────┼────────┼────────┼───────┼────────────┼──────────│
│  ☑ │ 张三 │ 95     │ 50     │ 1.2km │ ✅ 符合    │ [指定管理员]│
│  ☑ │ 李四 │ 88     │ 30     │ 2.5km │ ❌ 不符合  │           │
│  ☑ │ 王五 │ 92     │ 45     │ 0.8km │ ✅ 符合    │           │
│  ☐ │ 赵六 │ 75     │ 10     │ 3.0km │ ❌ 不符合  │           │
├─────────────────────────────────────────────────────────────┤

└─────────────────────────────────────────────────────────────┘
```

**分配操作**
1. 勾选入选的临工（根据需求人数）
2. 从符合条件的临工中指定一位为临时管理员
3. 可修改该订单的抽成比例（覆盖默认值）
4. 确认分配后，系统自动发送通知

### 5.3 订单详情

- 查看订单完整信息
- 查看参与临工列表
- 查看打卡记录、工时/计件记录
- 查看结算单
- 处理异常上报

## 8. 数据统计

### 8.1 用户数据

| 指标 | 说明 |
|-----|------|
| 新增用户数 | 按日/周/月统计 |
| 用户总数 | 企业用户/临工用户分类 |
| 认证通过率 | 认证通过数/提交数 |
| 活跃用户数 | 近7天/30天登录用户 |

### 8.2 业务数据

| 指标 | 说明 |
|-----|------|
| 发布信息数 | 按类型分类统计 |
| 查看联系方式次数 | 付费转化指标 |
| 用工订单数 | 按状态分类统计 |
| 用工完成率 | 完成订单/总订单 |

### 8.3 财务数据

| 指标 | 说明 |
|-----|------|
| 总收入 | 各收入项汇总 |
| 总支出 | 各支出项汇总 |
| 净收入 | 总收入 - 总支出 |

### 8.4 数据导出

- 所有统计数据支持导出Excel
- 支持按日/周/月筛选

## 9. 系统配置

### 9.1 可配置项汇总

**重要原则：所有数值类配置都在后台管理，代码中不写死**

| 配置分类 | 配置项 | 说明 |
|---------|-------|------|
| **价格配置** | 月会员价格 | 元 |
| | 年会员价格 | 元 |
| | 查看机会套餐 | 多个套餐，每个包含次数和价格 |
| | 广告位价格 | 每个广告位单独配置，元/天 |
| | 置顶价格 | 多个时长套餐，元/天 |
| **用工配置** | 默认抽成比例 | 百分比 |
| | 管理员服务费单价 | 元/人/小时 或 元/人/件 |
| | 超额报名比例 | 百分比 |
| | 出勤确认提醒时间 | 开工前N小时 |
| | 出勤确认截止时间 | 开工前N小时 |
| **用户配置** | 新用户初始查看机会 | 次数 |
| | 查看机会有效期 | 天数 |
| | 临工初始信用分 | 分数 |
| | 工厂初始信用分 | 分数 |
| | 管理员资格-最低接单数 | 单数 |
| | 管理员资格-最低信用分 | 分数 |
| **信息配置** | 默认信息有效期 | 天数 |
| | 审核模式 | 自动审核/人工审核 |
| **其他配置** | 会员到期提醒天数 | 提前N天提醒 |
| | 查看机会过期提醒天数 | 提前N天提醒 |
