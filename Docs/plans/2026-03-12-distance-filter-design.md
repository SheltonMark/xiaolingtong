# 临工首页按距离筛选功能设计

## 1. 功能概述

在临工首页添加按距离筛选招工信息的功能，帮助临工快速找到附近的工作机会。

## 2. 技术方案

### 2.1 地图服务

- **服务商**: 腾讯位置服务
- **Key**: NOWBZ-YXCL7-PMLX2-PJSVO-C52S2-CBFVW
- **SDK**: 微信小程序 JavaScript SDK v1.2
- **API**: calculateDistance (直线距离计算)

### 2.2 实现方式

1. **用户位置获取**
   - 用户点击距离筛选时才请求定位授权
   - 使用 `wx.getLocation()` 获取用户当前位置
   - 坐标系: GCJ02（腾讯地图使用的坐标系）

2. **距离计算**
   - 使用腾讯地图 SDK 的 `calculateDistance` 方法
   - 计算模式: `mode: 'straight'` (直线距离)
   - 批量计算列表中所有招工的距离

3. **距离筛选**
   - 筛选选项: 全部、1km内、3km内、5km内、10km内、10km以上
   - 根据用户选择过滤招工列表

4. **距离显示**
   - 招工卡片: 在地区和发布日期后显示"距您XXX"
   - 招工详情页: 在工作地点后显示"距您XXX"
   - 格式: < 1000米显示"XXX米"，≥ 1000米显示"X.Xkm"

## 3. 数据流

```
用户点击距离筛选
    ↓
请求定位授权
    ↓
获取用户位置 (latitude, longitude)
    ↓
加载招工列表
    ↓
批量计算每个招工的距离
    ↓
根据筛选条件过滤
    ↓
显示结果（卡片上显示距离）
```

## 5. 用户体验

### 5.1 定位授权流程

1. 用户首次点击距离筛选（非"全部"选项）
2. 显示"获取位置中..."加载提示
3. 请求定位授权
4. 授权成功: 计算距离并筛选
5. 授权失败: 显示提示弹窗，引导用户去设置开启权限

### 5.2 无定位权限处理

- 如果用户拒绝授权，自动重置筛选为"全部"
- 显示友好提示，说明需要位置权限才能使用距离筛选

### 5.3 距离显示规则

- 有位置权限且招工有坐标: 显示"距您XXX"
- 无位置权限或招工无坐标: 不显示距离

## 6. 数据库要求

招工信息需要存储经纬度字段：
- `latitude`: 纬度 (DECIMAL(10,7))
- `longitude`: 经度 (DECIMAL(10,7))

发布招工时通过地图选点获取坐标。
