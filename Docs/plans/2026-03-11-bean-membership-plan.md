## 三、会员权益规划

### 套餐价格（保持不变）

| 套餐     | 价格  | 时长  |
| -------- | ----- | ----- |
| 月度会员 | ¥99  | 30天  |
| 季度会员 | ¥238 | 90天  |
| 年度会员 | ¥799 | 365天 |

### 会员权益明细

| 权益                 | 非会员         | 会员                   |
| -------------------- | -------------- | ---------------------- |
| 每日免费查看联系方式 | 0次            | 5次/天（后台可配置）   |
| 超出后查看联系方式   | 10灵豆/次      | 5灵豆/次（5折）        |
| 帖子置顶             | 原价           | 8折                    |
| ~~帖子刷新~~       | ~~5灵豆/次~~ | ~~免费3次/天~~       |
| VIP标识              | 无             | 昵称旁显示橙色VIP图标  |
| 置顶标识             | 无             | 发布的帖子带"会员"标签 |
| 广告投放折扣         | 原价           | 9折                    |

> 核心逻辑：会员 = 每日免费额度 + 超出打折 + 身份标识。不是"无限免费"，这样灵豆大额套餐对重度用户仍有价值。

## 五、广告系统

### Banner 广告

- 位置：首页顶部轮播
- 素材：纯图片，750×300px（5:2 比例）
- 价格：¥200/天（后台可配置）
- 审核：后台管理员审核通过后展示
- 入口：企业端"我的"页面 → 功能入口格子 → 广告投放

### 展示逻辑

- 有审核通过的广告时：展示广告图片轮播
- 无广告时：展示平台默认运营 banner（当前的渐变占位图）
- 广告图片右下角带半透明"广告"标签
